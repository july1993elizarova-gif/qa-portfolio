Тестовое задание – Тестирование UI и конфигуратора компонента

Описание задания

Работодатель предоставил два задания:
1. Написать тест-кейсы для формы регистрации в закрытый клуб тестировщиков.
2. Составить чек-лист функционального и нефункционального тестирования окна "Конфигуратор компонента".

Цель задания — проверить навыки тест-дизайна, анализа требований и составления тестовой документации.

Задание 1 – Тест-кейсы для формы регистрации

Форма регистрации имеет следующие правила:
- Регистрация доступна только тестировщикам
- Минимальный возраст пользователя — 18 лет
- Email является обязательным полем
- Чекбокс подписки на новости — необязательный

В тестировании использованы техники тест-дизайна:
- Анализ граничных значений
- Классы эквивалентности
- Позитивные и негативные сценарии

Всего написано: 14 тест-кейсов

Задание 2 – Чек-лист тестирования конфигуратора компонента

Проверялось окно конфигурации компонента, в котором можно:
- добавлять опции
- удалять опции
- изменять порядок опций
- использовать поиск
- просматривать свойства компонента

Чек-лист включает:
Функциональные проверки:
- добавление опций
- удаление опций
- изменение порядка
- фильтрацию
- отображение свойств

Нефункциональные проверки:
- производительность
- стабильность
- UI
- UX
