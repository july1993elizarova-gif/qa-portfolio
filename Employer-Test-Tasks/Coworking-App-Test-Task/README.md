Тестовое задание — мобильное приложение коворкингов

В рамках тестового задания было протестировано мобильное приложение для поиска и бронирования коворкингов.

Были составлены чек-листы для следующих разделов:

- Регистрация и авторизация
- Главный экран
- Новости
- Список коворкингов
- Бронирование
- Карта коворкингов
- Банковские карты

Цель: проверить навыки составления тестовой документации и анализа функциональности мобильного приложения.
