Бронирование:
Нажать на кнопку «Забронировать» у выбранного коворкинга / Открывается BottomSheet с формой бронирования
Проверить ввод даты и времени / Поля активны, открываются и позволяют выбрать значения
Проверить ввод комментария / Можно ввести до символов текста
Проверить ввод промокода / Поле активно, принимает не более символов
Проверить отображение цены / После выбора даты и времени отображается цена (или анимация загрузки "уточняем цену")
Проверить кнопку «Забронировать» / Кнопка активируется после заполнения обязательных полей
Проверить успешное бронирование / При корректных данных появляется тост "Коворкинг забронирован"
Проверить ошибку при занятом времени / Появляется сообщение «Время занято, выберите другое»
Проверить ошибку при недоступности брони / Отображается «Упс, похоже это время уже занято»
Проверить корректность закрытия формы бронирования / закрывается по кнопке «назад» или свайпу вниз
Проверить отображение истории бронирований после успешной операции / Новая бронь появляется в разделе “Актуальное бронирование”
