Отображение при разных состояниях:
Нет карт / При отсутствии карт отображается заглушка "Упс, тут ещё ничего нет" и кнопка "Добавить карту"
1-2 карты / Отображается список карт с кнопкой "Добавить карту" внизу экрана
3 карты / Отображается список из 3 карт, кнопка "Добавить карту" скрыта
Выделение главной карты / Главная карта визуально выделена

Добавление карт
Кнопка "Добавить карту" / При нажатии открывается экран/форма добавления новой карты
Лимит карт / Невозможно добавить более 3 карт (кнопка скрывается после добавления третьей карты)

Управление картами
Смена главной карты / При тапе на неглавную карту появляется алерт "Сменить главную карту" с кнопками подтверждения/отмены
Удаление карты / При долгом нажатии на карту появляется алерт "Удалить карту?" с кнопками подтверждения/отмены
Удаление главной карты / Проверить сценарий удаления главной карты (какая карта становится главной после удаления)
