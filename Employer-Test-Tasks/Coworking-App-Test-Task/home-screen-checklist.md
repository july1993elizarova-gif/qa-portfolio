Раздел Новости:
Открыть приложение и перейти на экран главная / Экран открывается без ошибок, интерфейс отображается корректно
Проверить наличие разделов: Новости, Лучший коворкинг, Описание / Все разделы присутствуют на экране
Проверить отображение новостей / отображаются 3 последние новости
Проверить корректность контента новостей / Новости содержат изображение, заголовок, краткий текст
Нажать на новость / Открывается окно (экран) с полной новостью

Раздел “Лучший коворкинг”
Проверить расчет рейтинга коворкингов / В блок «Лучший коворкинг» попадают 3 коворкинга с наилучшим результатом по формуле (рейтинг / количество отзывов)
Проверить переход по карточке коворкинга / При нажатии открывается детальная страница коворкинга

Раздел “Перейти в список”
Проверить кнопку «Перейти в список» / При нажатии открывается экран Список коворкингов

Раздел “Описание”
Проверить наличие текста описания / Отображается текст о коворкинге
Проверить корректность форматирования текста / Текст читаемый, переносы корректные, не выходит за рамки блока
Проверить цвет и фон блока описания / Соответствует дизайну (градиент фона, контраст текста)
