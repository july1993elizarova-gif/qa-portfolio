Отображение карты и интерфейса:
Загрузка экрана / Экран карты открывается без ошибок при переходе из меню.
Отображение карты / Карта отображается корректно, без пустых областей или артефактов.
Элементы управления / Присутствуют стандартные элементы управления картой (зум, перемещение).

Отображение коворкинга на карте:
Маркеры/ Иконки коворкингов / Коворкинги отображаются на карте в виде маркеров/иконок.
Корректность расположения / Маркеры расположены по правильным географическим координатам (не все в одной точке).
Кол-во маркеров / На карте отображается ожидаемое количество коворкингов (соответствует данным).

Взаимодействие с маркерами коворкингов:
Тап на маркер / При нажатии на иконку коворкинга происходит действие (появляется балун/инфо-окно с названием или открывается bottomsheet с детальной информацией).
Переход на карточку / При тапе на кнопку в балуне или на сам маркер (если балуна нет) происходит переход на экран деталей коворкинга.

Навигация и доп функции:
Кнопка перехода в список / На экране присутствует кнопка для перехода в общий список коворкингов.
Работа кнопки списка / При нажатии на кнопку происходит переход на экран "Список коворкингов".
