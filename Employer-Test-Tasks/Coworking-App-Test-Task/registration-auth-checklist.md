Чек-лист — Регистрация и авторизация

Регистрация

- ввод валидных данных → пользователь зарегистрирован
- регистрация уже зарегистрированного пользователя → регистрация запрещена
- регистрация удаленного пользователя
- все поля валидны, логин невалидный → регистрация невозможна
- все поля валидны, email невалидный → регистрация невозможна

Авторизация

- ввод валидных данных → пользователь входит в личный кабинет
- многократное нажатие на кнопку "Войти" → кнопка не блокируется
- пустые поля
- одно поле пустое
- одно поле содержит невалидные данные
- оба поля невалидные
- ввод пробелов и невидимых символов
- отображение ошибки при некорректных данных
- скрытие / отображение пароля
