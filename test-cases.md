# Test Cases

## Test Case 1

**ID:** TC-01  
**Title:** Проверка открытия главной страницы  

**Preconditions:**  
Пользователь открывает сайт

**Steps:**
1. Открыть браузер
2. Перейти на главную страницу сайта

**Expected result:**  
Главная страница успешно загружается
